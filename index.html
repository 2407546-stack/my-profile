What is HTML?

HTML stands for HyperText Markup Language. It is used to create and structure web pages.

Structural Elements

<header> – Defines the top section of a page.

<section> – Groups related content.

<footer> – Defines the bottom of a page.

Text Formatting Elements

<b> – Makes text bold.

<i> – Italic text.

<strong> – Important text.

Lists

<ul> – Unordered list.

<ol> – Ordered list.

Links

<a> – Used to create hyperlinks.

Images

<img> – Displays an image on a webpage.

Tables

<table> – Organizes data into rows and columns.

Forms

<form> – Collects user input.

Semantic Elements

<article> – Represents independent content.

<nav> – Navigation links.
